1. 利用教材定理 6.12 证明中的方法通过构造自动机来判定下面两个句子是否正确

   1. forall x exist y[x+x=y] 

      将变量 $x$ 替换为 $x_1$，$y$ 替换为 $x_2$。
      原句子 $\phi = \forall x_1 \exists x_2 [x_1 + x_1 = x_2]$。

      根据定理，设 $l=2$，我们定义一系列公式：

      *   **$\phi_2$** = $x_1 + x_1 = x_2$  (不含量词的核心公式)
      *   **$\phi_1$** = $\exists x_2 \phi_2 = \exists x_2 [x_1 + x_1 = x_2]$
      *   **$\phi_0$** = $\forall x_1 \phi_1 = \forall x_1 \exists x_2 [x_1 + x_1 = x_2]$  (即原句子 $\phi$)

      $A_2$ 是一个判定两个变量关系的自动机。

      *   **字母表 $\Sigma_2$**: 包含表示两个二进制位对的符号，即 $\Sigma_2 = \left\{ \begin{bmatrix} 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \end{bmatrix}, \begin{bmatrix} 1 \\ 0 \end{bmatrix}, \begin{bmatrix} 1 \\ 1 \end{bmatrix} \right\}$。顶部的位属于 $x_1$，底部的位属于 $x_2$。
      *   **读取顺序**: 自动机从**最高有效位 (MSB) 到最低有效位 (LSB)** 读取二进制串。
      *   **状态设计**: $x_1 + x_1 = x_2$ 等价于 $2x_1 = x_2$。这意味着 $x_2$ 的二进制表示就是 $x_1$ 左移一位（末尾补0）。在从左向右（MSB优先）读取时，我们跟踪两者的差值预期。设 $D = 2 \cdot (\text{已读 } x_1 \text{ 的值}) - (\text{已读 } x_2 \text{ 的值})$。
          *   读取新符号 $\begin{bmatrix} b_1 \\ b_2 \end{bmatrix}$ 后，新的差值为 $D' = 2D + 2b_1 - b_2$。
          *   为了使等式最终成立，自动机结束时必须有 $D=0$。
          *   分析可得，有效状态只有 $D=0$ 和 $D=-1$。如果差值超出这个范围，将无法恢复到 0。
      *   **定义 DFA $A_2$**:
          *   状态集 $Q_2 = \{q_0, q_{-1}, q_{reject}\}$ (分别对应差值 0, -1, 和拒绝状态)
          *   起始状态: $q_0$
          *   接受状态集 $F_2 = \{q_0\}$
          *   转移函数 $\delta_2$:
              *   $\delta_2(q_0, \begin{bmatrix} 0 \\ 0 \end{bmatrix}) = q_0$  (2*0 + 2*0 - 0 = 0)
              *   $\delta_2(q_0, \begin{bmatrix} 0 \\ 1 \end{bmatrix}) = q_{-1}$ (2*0 + 2*0 - 1 = -1)
              *   $\delta_2(q_0, \begin{bmatrix} 1 \\ 0 \end{bmatrix}) = q_{reject}$ (2*0 + 2*1 - 0 = 2，超出范围)
              *   $\delta_2(q_0, \begin{bmatrix} 1 \\ 1 \end{bmatrix}) = q_{reject}$ (2*0 + 2*1 - 1 = 1，超出范围)
              *   $\delta_2(q_{-1}, \begin{bmatrix} 0 \\ 0 \end{bmatrix}) = q_{reject}$ (2*(-1) + 2*0 - 0 = -2)
              *   $\delta_2(q_{-1}, \begin{bmatrix} 0 \\ 1 \end{bmatrix}) = q_{reject}$ (2*(-1) + 2*0 - 1 = -3)
              *   $\delta_2(q_{-1}, \begin{bmatrix} 1 \\ 0 \end{bmatrix}) = q_0$  (2*(-1) + 2*1 - 0 = 0)
              *   $\delta_2(q_{-1}, \begin{bmatrix} 1 \\ 1 \end{bmatrix}) = q_{-1}$ (2*(-1) + 2*1 - 1 = -1)
              *   对于 $q_{reject}$，任何输入都停留在 $q_{reject}$。

      存在量词 $\exists$ 的构造方法是将确定性自动机转换为非确定性自动机(NFA)，通过“猜测”被量化变量($x_2$)的值来实现。

      *   **字母表 $\Sigma_1$**: $\{[0], [1]\}$ (仅包含 $x_1$ 的位)。
      *   **基础转移猜测**: 将 $A_2$ 中 $x_2$ 的位隐去，变为非确定性转移。例如，在 $q_0$ 读入 $[0]$ 时，可能对应 $\begin{bmatrix} 0 \\ 0 \end{bmatrix}$ (去 $q_0$) 或 $\begin{bmatrix} 0 \\ 1 \end{bmatrix}$ (去 $q_{-1}$)，所以转移到 $\{q_0, q_{-1}\}$。
      *   **处理前导0 (Initially规则)**:由于输入字符串对齐问题，$A_1$ 需要在读取输入前，非确定性地猜测 $x_2$ 较长时产生的前导 0。
          *   新增起始状态 $s_1$。
          *   从 $s_1$ 添加 $\epsilon$-转移到 $A_2$ 的起始状态($q_0$)通过仅由 $x_1$ 为 0 的符号构成的字符串所能到达的所有状态。
          *   从 $q_0$ 出发，读取 $\begin{bmatrix} 0 \\ 0 \end{bmatrix}$ 或 $\begin{bmatrix} 0 \\ 1 \end{bmatrix}$ 的组合，可到达的状态有 $\{q_0, q_{-1}, q_{reject}\}$。
          *   因此，$s_1 \xrightarrow{\epsilon} \{q_0, q_{-1}, q_{reject}\}$。
      *   **NFA $A_1$ 的接收状态**: 同 $A_2$，即 $\{q_0\}$。
      *   **$A_1$ 的等价 DFA 分析**:
          为了方便后续取反操作，分析一下 $A_1$ 接受的语言。
          *   $A_1$ 包含 $\epsilon$-转移，因此从开始状态 $s_1$ 可以直接到达 $\{q_0, q_{-1}, q_{reject}\}$。由于包含接受状态 $q_0$，空串是被接受的。
          *   无论输入是 $[0]$ 还是 $[1]$，从子集 $\{q_0, q_{-1}, q_{reject}\}$ 出发，结合我们前面推导的转移，其下一个状态的子集总是包含 $q_0$。
          *   这意味着 NFA $A_1$ 不管读入什么关于 $x_1$ 的二进制串，总存在一条路径能到达接受状态 $q_0$。
          *   **结论**: $A_1$ 接受 $\Sigma_1^*$ 上的所有字符串。这在逻辑上是吻合的：因为对于**任意**自然数 $x_1$，总**存在**一个自然数 $x_2$ 使得 $x_1 + x_1 = x_2$ (即 $x_2 = 2x_1$)。

      $\forall x_1 \phi_1$ 等价于 $\neg \exists x_1 \neg \phi_1$。因此，需要执行三次操作：取反 -> 构造 $\exists$ 自动机 -> 再次取反。

      *   **步骤 4a: 构造 $\neg \phi_1$ 的自动机 ($\overline{A_1}$)**
          *   由于 $A_1$ 接受 $\Sigma_1$ 上的**所有**字符串，它的补集自动机 $\overline{A_1}$ **不接受任何**字符串。
          *   其语言 $L(\overline{A_1}) = \emptyset$。
      *   **步骤 4b: 构造 $\exists x_1 (\neg \phi_1)$ 的自动机**
          *   我们基于 $\overline{A_1}$ 构造一个 NFA 来猜测 $x_1$ 的位。
          *   由于 $\overline{A_1}$ 不包含任何接受状态（或者无论怎么猜测转移都无法到达接受状态），这个新的 NFA 也同样**不接受任何**字符串。
          *   其语言为 $\emptyset$。
      *   **步骤 4c: 构造最终的 $A_0$ (再次取反)**
          *   $A_0$ 是上一步自动机的补集。
          *   因为上一步的自动机接受空集，取反后，$A_0$ 接受其字母表 $\Sigma_0 = \{[]\}$ 上的**所有**字符串。
          *   因此， $L(A_0) = \Sigma_0^*$。

      根据我们的构造，$A_0$ 接受所有字符串（包括空串 $\epsilon$）。
      因此，**自动机 $A_0$ 接受 $\epsilon$**。

      **结论**：利用该方法构造的自动机判定句子 $\forall x \exists y [x + x = y]$ 是**正确的 (True)**。

   2. exist y forall x [x+x=y]

      按照从左到右的顺序，将变量 $y$ 替换为 $x_1$，$x$ 替换为 $x_2$。
      原句子 $\phi = \exists x_1 \forall x_2 [x_2 + x_2 = x_1]$。

      根据定理，设 $l=2$，我们定义一系列公式：

      *   **$\phi_2$** = $x_2 + x_2 = x_1$  (不含量词的核心公式)
      *   **$\phi_1$** = $\forall x_2 \phi_2 = \forall x_2 [x_2 + x_2 = x_1]$
      *   **$\phi_0$** = $\exists x_1 \phi_1 = \exists x_1 \forall x_2 [x_2 + x_2 = x_1]$  (即原句子 $\phi$)

      $A_2$ 是一个判定两个变量关系的确定性有穷自动机 (DFA)。

      *   **字母表 $\Sigma_2$**: 包含表示两个二进制位对的符号，即 $\Sigma_2 = \left\{ \begin{bmatrix} 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \end{bmatrix}, \begin{bmatrix} 1 \\ 0 \end{bmatrix}, \begin{bmatrix} 1 \\ 1 \end{bmatrix} \right\}$。约定**顶部的位属于 $x_1$，底部的位属于 $x_2$**。
      *   **读取顺序**: 自动机从**最高有效位 (MSB) 到最低有效位 (LSB)** 读取二进制串。
      *   **状态设计**: $x_2 + x_2 = x_1$ 等价于 $x_1 = 2x_2$。我们在从左向右读取时，跟踪当前已读部分的差值 $D = V_{x1} - 2V_{x2}$。
          *   读取新符号 $\begin{bmatrix} b_1 \\ b_2 \end{bmatrix}$ 后，新差值为 $D' = (2V_{x1} + b_1) - 2(2V_{x2} + b_2) = 2(V_{x1} - 2V_{x2}) + b_1 - 2b_2 = 2D + b_1 - 2b_2$。
          *   经过分析，为了使差值最终可能回到0，有效的状态只能是 $D \in \{0, 1, -1\}$。超出此范围的差值将进入一个“拒绝”死胡同状态 $q_{rej}$。
      *   **定义 DFA $A_2$**:
          *   状态集 $Q_2 = \{q_0, q_1, q_{-1}, q_{rej}\}$ (分别对应差值 0, 1, -1, 和拒绝)
          *   起始状态: $q_0$
          *   接受状态集 $F_2 = \{q_0\}$ (必须完全相等)
          *   部分关键转移函数 $\delta_2$:
              *   $\delta_2(q_0, \begin{bmatrix} 0 \\ 0 \end{bmatrix}) = q_0$
              *   $\delta_2(q_0, \begin{bmatrix} 1 \\ 0 \end{bmatrix}) = q_1$ (读入 $x_1$ 的1，差值变大)
              *   $\delta_2(q_0, \begin{bmatrix} 0 \\ 1 \end{bmatrix}) = q_{rej}$ (差值变为-2，无法恢复)
              *   $\delta_2(q_1, \begin{bmatrix} 0 \\ 1 \end{bmatrix}) = q_0$ (差值为1时，$x_2$ 读入1，恰好抵消：2(1)+0-2(1)=0)
              *   任何到达 $q_{rej}$ 的转移都将停留在 $q_{rej}$。

      全称量词 $\forall x_2 \phi_2$ 等价于 $\neg \exists x_2 \neg \phi_2$。因此需要三个步骤：取反、构造存在量词自动机、再次取反。

      * **步骤 3a: 构造 $\neg \phi_2$ 的自动机 ($\overline{A_2}$)**

        *   将 DFA $A_2$ 的接受状态与非接受状态互换。
        *   $\overline{A_2}$ 的接受状态变为 $\{q_1, q_{-1}, q_{rej}\}$。

      * **步骤 3b: 构造 NFA $A_1'$ 对应 $\exists x_2 (\neg \phi_2)$**

        *   字母表 $\Sigma_1 = \{[0], [1]\}$ (仅包含 $x_1$ 的位)。
        *   **投影猜测**: 将 $\overline{A_2}$ 中 $x_2$ 的位隐去，变为非确定性转移。
        *   **关键分析**: 考虑从状态 $q_0$ 读取符号 $[0]$ (即 $x_1=0$) 时：
            *   如果猜测 $x_2=0$，相当于读取 $\begin{bmatrix} 0 \\ 0 \end{bmatrix}$，转移到 $q_0$。
            *   如果猜测 $x_2=1$，相当于读取 $\begin{bmatrix} 0 \\ 1 \end{bmatrix}$，在 $A_2$ 中这会转移到 $q_{rej}$。
            *   因此，在 NFA $A_1'$ 中，$\delta'_{1}(q_0, [0]) = \{q_0, q_{rej}\}$。
        *   **处理前导0 (Initially规则)**: $A_1'$ 包含一个新的起始状态 $s_1$，它通过 $\epsilon$-转移连接到仅读取 $x_1=0$ 时能到达的状态。
            *   即通过符号 $\begin{bmatrix} 0 \\ 0 \end{bmatrix}$ 和 $\begin{bmatrix} 0 \\ 1 \end{bmatrix}$ 从 $q_0$ 可达的状态，也就是 $\{q_0, q_{rej}\}$。
            *   所以有 $\epsilon$-转移：$s_1 \xrightarrow{\epsilon} \{q_0, q_{rej}\}$。
        *   **$A_1'$ 的语言**: 注意到 $q_{rej}$ 在 $A_1'$ 中是**接受状态**。而且一旦进入 $q_{rej}$，任何输入都会让它停留在 $q_{rej}$。由于一开始通过 $\epsilon$-转移就能直接进入接受状态 $q_{rej}$，这意味着 $A_1'$ 接受空串 $\epsilon$ 以及其字母表上的**所有**字符串。即 $L(A_1') = \Sigma_1^*$。

      * **步骤 3c: 构造最终的 $A_1$ (再次取反)**

        * $A_1$ 是 $A_1'$ 的补集自动机。

        * 因为 $A_1'$ 接受所有字符串，所以 $A_1$ **不接受任何字符串**（其接受语言为空集 $\emptyset$）。

          

      - 要基于 $A_1$ 构造 $A_0$，通过非确定性地猜测 $x_1$ 的位来实现 $\exists x_1$。

      *   但是，无论我们在 $\Sigma_0$ ($=\{[]\}$) 上猜测 $x_1$ 生成什么样的比特流，这些流将被送入 $A_1$ 进行判定。
      *   在上一步已经得出结论：**$A_1$ 不接受任何字符串**，它的内部没有任何可以到达接受状态的有效路径。
      *   因此，新的 NFA $A_0$ 也将没有任何路径可以到达接受状态。
      *   **结论**：$A_0$ 不接受任何字符串，其语言为 $\emptyset$。

      由于推导得出自动机 $A_0$ 不接受任何字符串（包括空串 $\epsilon$），它将拒绝该输入。

      **判定结果**：利用该方法构造的自动机判定句子 $\exists y \forall x [x + x = y]$ 是**错误的 (False)**。

2. 通过阅读 Godel numbering 或者 arithmetization of Syntax 的知识，给出 Lemma 6.14 的详细证明，即给出$phi_{M,w}$的具体构造

   我们要构造一个公式 $\phi_{M,w}(x)$，它的直观含义是：**“数字 $x$ 编码了图灵机 $M$ 在输入 $w$ 上的一个合法的、最终进入接受状态的计算历史 (Accepting Computation History)。”**

   一个计算历史是一个构型 (Configuration) 的序列：$C_0, C_1, C_2, \dots, C_T$。

   *   $C_0$ 是初始构型。
   *   $C_i \to C_{i+1}$ 符合 $M$ 的转移函数。
   *   $C_T$ 包含接受状态 $q_{acc}$。

   为了用算术公式表示这些，我们需要完成以下三步：

   1.  **编码 (Encoding)**：定义如何将符号、构型和整个计算历史映射到一个自然数 $x$。
   2.  **算术工具 (Arithmetic Tools)**：构造基本的公式，用于从数字 $x$ 中“提取”和“比较”特定的符号。
   3.  **约束构造 (Constraint Construction)**：用上述工具写出检查初始状态、合法转移和接受状态的公式。

   ---

   设图灵机 $M = (Q, \Sigma, \Gamma, \delta, q_0, q_{acc}, q_{rej})$。
   定义一个包含了所有可能出现在计算历史字符串中的符号的字母表：
   $A = Q \cup \Gamma \cup \{\#\}$
   其中 $\#$ 用作构型之间的分隔符。

   假设字母表 $A$ 中总共有 $k$ 个符号。我们给每个符号分配一个唯一的、大于 0 的整数代码：
   $enc: A \to \{1, 2, \dots, k\}$

   **字符串的编码：**
   为了方便用算术提取字符，我们将计算历史字符串视为一个**大基数 $B$ 下的整数**。
   令基数 $B = k + 1$。
   一个字符串 $s = s_1 s_2 \dots s_L$ （其中 $s_i \in A$）被编码为一个自然数 $x$：
   $$x = \sum_{i=1}^{L} enc(s_i) \cdot B^{i-1}$$

   ---

   我们需要一个公式 $At(x, i, c)$，表示“在编码为 $x$ 的字符串中，第 $i$ 个位置（从 1 开始计）的符号的代码是 $c$”。

   在基数 $B$ 下，数字 $x$ 的第 $i$ 位数字是 $c$，等价于：存在商 $q$ 和余数 $r$，使得
   $$x = q \cdot B^i + c \cdot B^{i-1} + r$$
   并且满足 $0 \le c < B$ 且 $0 \le r < B^{i-1}$。

   用数论语言（仅使用 $+$, $\times$, $=$）写出 $At(x, i, c)$：
   $$At(x, i, c) \equiv \exists q \exists r [ x = q \cdot B^i + c \cdot B^{i-1} + r \land c < B \land r < B^{i-1} ]$$

   ---

   为了使转移检查变得局部且易于算术化，假设计算历史中**所有的构型都被填充 (padded) 到了相同的长度 $S$**（$S$ 是计算过程中使用的最大空间）。空白处用空格符 $\sqcup$ 填充。

   一个合法的计算历史字符串如下所示：
   $$\# C_0 \# C_1 \# \dots \# C_T \#$$

   *   每个 $C_m$ 长度刚好为 $S$。
   *   分隔符 $\#$ 周期性出现，总长度 $L = (T+1)(S+1) + 1$。
   *   相邻构型的对应字符（例如 $C_m$ 的第 $j$ 个字符和 $C_{m+1}$ 的第 $j$ 个字符）在整个字符串中的距离恰好是 $S+1$。

   公式 $\phi_{M,w}(x)$ 将形如：
   $\exists L \exists S \exists T [ \phi_{format} \land \phi_{start} \land \phi_{accept} \land \phi_{transition} ]$

   检查字符串两端是 $\#$，并且每隔 $S$ 个字符有一个 $\#$。
   $$\phi_{format} \equiv At(x, 1, enc(\#)) \land At(x, L, enc(\#)) \land$$
   $$\forall i \le L \Big[ At(x, i, enc(\#)) \leftrightarrow \exists m \le T (i = m(S+1) + 1) \Big]$$

   检查 $C_0$ 是否是 $q_0 w_1 w_2 \dots w_n \sqcup \dots \sqcup$。
   设输入 $w$ 的长度为 $n$。这里的 $enc(q_0)$ 和 $enc(w_1) \dots$ 都是具体的常数数字。
   $$\phi_{start} \equiv At(x, 2, enc(q_0)) \land At(x, 3, enc(w_1)) \land \dots \land At(x, n+2, enc(w_n)) \land$$
   $$\forall i [ (n+3 \le i \le S+1) \to At(x, i, enc(\sqcup)) ]$$

   字符串中必须至少包含一个 $q_{acc}$ 符号。
   $$\phi_{accept} \equiv \exists i \le L [ At(x, i, enc(q_{acc})) ]$$

   图灵机的转移是局部的。如果在 $C_{m+1}$ 中观察某个位置 $j$ 的符号，它**唯一地**由 $C_m$ 中对应位置及其左右相邻的三个符号决定。
   这被称为 $2 \times 3$ 窗口检查。

   定义一个谓词 $Yields(c_1, c_2, c_3, c_{new})$，表示如果前一个构型的连续三个字符是 $c_1 c_2 c_3$，根据 $M$ 的转移规则 $\delta$，下一个构型正下方的字符必须是 $c_{new}$。
   由于 $M$ 是固定的，所有合法的 $(c_1, c_2, c_3, c_{new})$ 组合是一个**有限集合**。
   因此，$Yields$ 可以写成一个巨大的有限析取式（OR 语句）：
   $$Yields(c_1, c_2, c_3, c_{new}) \equiv$$
   $$[(c_1=a \land c_2=b \land c_3=c \land c_{new}=d) \lor \dots \text{所有合法组合} \dots]$$

   现在，我们强制要求字符串中每一个不属于 $C_0$ 且不是 $\#$ 的符号，都合法地从其上一个构型继承而来。
   对于 $C_{m+1}$ 中的索引 $i$（$i > S+1$ 且 $At(i)$ 不是 $\#$），它对应的 $C_m$ 中的中心位置是 $i - (S+1)$。

   $$\phi_{transition} \equiv \forall i [ (S+2 \le i \le L-1 \land \neg At(x, i, enc(\#))) \to$$
   $$\exists c_1, c_2, c_3, c_{new} \Big($$
   $$At(x, i - (S+1) - 1, c_1) \land$$
   $$At(x, i - (S+1), c_2) \land$$
   $$At(x, i - (S+1) + 1, c_3) \land$$
   $$At(x, i, c_{new}) \land$$
   $$Yields(c_1, c_2, c_3, c_{new})$$
   $$\Big) ]$$

   ---

   最终的公式 $\phi_{M,w}(x)$ 就是将上述所有部分组装在一起：
